# 🤖 AI Course Assistant Chatbot

## 📌 Problem
Students in AI courses often struggle to understand complex concepts.

## 👤 User Persona
- University students learning AI  
- Need simple explanations  
- Want quick answers  

## 💡 Solution
A chatbot that answers AI-related questions.

## ⚙️ Features
- AI question answering  
- Simple explanations  
- Interactive chatbot  

## ▶️ How to Run
pip install -r requirements.txt  
streamlit run app.py  

## 🧪 Demo Explanation
This chatbot is a prototype AI assistant designed to help students understand AI concepts.

Due to API quota limitations, responses are simulated to demonstrate how a real AI chatbot would function.
